## 项目说明  
此项目研发的是一款类似网易云音乐的音乐小程序，将实现基于网易云APP端的功能复制及创新，目前正在研发中。
## 运行说明
1. 在已安装node的前提下，在`/NeteaseCloudMusicApi`目录下执行`node app.js`启动服务器；
2. 使用微信开发者工具打开`BX Music`即可;
**若要实现真机调试，则需要内网穿透，推荐使用natapp实现**
